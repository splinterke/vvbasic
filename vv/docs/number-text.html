NUMBER and TEXT

VVBasic allows you to create variables and give them values. A variable is a name for either a number or a text string. Here are some examples:

VVBasic commands Output
NUMBER age = 12
TEXT name = "Sue"
PRINT "The age of " name " is " age  		The age of Sue is 12 

The first two lines of this little VVBasic program create two variables. The variable age is a number with the value 12 and the variable name is a text string which has the value Sue. The PRINT command on the third line displays The age of Sue is 12. In a PRINT command, a variable is replaced by its value, so the computer replaces name with Sue and age with 12.

Variable names must begin with a letter, and can only be made of letters, numbers, and the underline character _. Here are some examples of good and bad variable names:

Good Bad
AGE 1DERFUL
Choice User'sChoice
DAY_OF_WEEK DAY OF WEEK
FavoriteColor Favorite-Color

Text variables can only contain text strings, and number variables can only contain numbers. A variable can be created without a value, as in these examples:

VVBasic commands
NUMBER item_price    
TEXT item_name

If a variable does not have a value, it must be given one with a LET command or an INPUT command before it is printed.
