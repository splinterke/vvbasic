Built in functions

A number of functions are built in to VVBasic. A function transforms one or more values into another value. For instance, suppose you want to change "Hello" to another text string with the same letters, all in upper case. You can use the ToUpper function to do this:

VVBasic commands Output
PRINT "Hello", ToUpper("Hello")   Hello HELLO  

The information inside the parentheses ("Hello" in the example above) is called the argument or arguments. The value of the function, based on the argument, is called the function's return value. A function creates a return value, but it doesn't change its argument.

VVBasic commands Output
TEXT Name = "George Washington" 
PRINT ToUpper(Name)
PRINT Name
GEORGE WASHINGTON 
George Washington

In the above example, notice that the value of the Name variable was not changed by ToUpper.
